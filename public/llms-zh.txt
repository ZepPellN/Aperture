# Aperture

> 一个面向 AI Agent 的 Markdown-first 个人 Wiki 系统。
> 将原始笔记、链接、摘录和日常日志转换为可浏览的知识图谱，保留来源出处、语义关联、演化历史，并为 AI 提供结构化 API。

## 这是什么

Aperture 是一个 Next.js 查看器，从文件系统读取纯 Markdown 文件，并将其编译为结构化知识系统。
它不是笔记应用，而是一个透镜：你像往常一样写 Markdown，Aperture 将其渲染为你和你的 Agent 可以探索的东西。

## 核心能力

### Wiki 系统
- `/wiki/<slug>` 文章页面：渲染后的 Markdown、wikilink 解析、来源追踪、演化时间线、反向链接、语义邻居路径。
- 首页：关键词搜索 + 语义扩展搜索、分类浏览、最近更新时间线、统计栏。
- 来源出处：每篇文章追踪其思想来源（内联链接、吸收日志、frontmatter、手动贡献），并标注贡献度等级。

### 图谱探索（4 种视图）
- **Network Graph** — Sigma.js WebGL 力导向图。聚焦模式、集群高亮、相机动画。
- **Topo Map** — Canvas 等高线密度地形图，D3 contour。分类色晕、悬浮提示。
- **Semantic Map** — 同等视觉语言，但使用 UMAP 语义坐标而非力导向布局。
- **Nest Graph** — React Three Fiber 3D 有机集群。种子形状节点、生长动画边、轨道控制。

### 语义基础设施
- 基于文本嵌入的 UMAP 降维。
- 余弦相似度邻居计算，用于语义路径和随机漫步。
- DBSCAN 聚类，在 `/clusters` 发现知识岛屿。

### 生活仪表盘
- `/life` 将每日日记、每周意图、习惯追踪和目标列表编译为统一仪表盘。
- 每周统计、意图卡片（STOP/START/FORGIVE/SELF-CARE）、下一步行动、心情时间线、习惯热力图、任务速度图、生活领域、周回顾。

### Agent API
- `/api/wiki/<slug>` — JSON API，返回 slug、标题、分类、Markdown、HTML、来源、反向链接、语义邻居、阅读时间、字数。
- `/llms.txt` 和 `/llms-full.txt` — 供 Agent 阅读的系统描述。
- `.agents/skills/` 中捆绑的 Claude Code skills，用于 Wiki 维护。

## 核心路由

- `/` — 首页：搜索、分类、最近更新、统计
- `/wiki/<slug>` — 文章页（来源、演化、反向链接、迷你图）
- `/graph` — 全屏图谱浏览器（4 种视图）
- `/clusters` — 语义聚类浏览器
- `/walk` — 语义邻居随机漫步
- `/life` — 个人生活仪表盘
- `/api/wiki/<slug>` — 文章数据 JSON API

## 数据格式

- 源文件：带 YAML frontmatter 的纯 Markdown。
- Wikilink 语法：`[[slug|label]]`。
- 生活数据：`witness/` 和 `life-wiki/` 目录中的自定义 Markdown 格式。
- 无数据库。无专有格式。文件优先。

## Agent 接入

让 Agent 读取 `AGENT_SETUP.md` 即可完成全套部署：仓库脚手架、技能安装、内容摄取、启动查看器。
