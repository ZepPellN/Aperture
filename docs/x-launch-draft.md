```text
I’ve been building Aperture: a markdown-first wiki viewer that turns a local vault into a knowledge graph for humans and AI agents.

It reads plain files, then adds:

- source provenance on every article
- backlinks + local graph context
- graph / cluster / semantic walk views
- llms.txt + JSON APIs for agents
- upgraded wiki skills for atomic notes, MOCs, and Zettelkasten-style maintenance

The point isn’t another note-taking app.
It’s compounding context: keep writing markdown, let agents maintain the structure, and give future sessions a system they can actually read.

Repo: https://github.com/ZepPellN/Aperture
```
