# Launch Thread

Aperture turns a folder of markdown into a navigable knowledge graph for humans and agents.

## Draft angle

Most note systems optimize for capture. Aperture optimizes for compounding context: source provenance, semantic trails, MOCs, atomic notes, backlinks, and agent-readable APIs.

## Demo beats

1. Open the dashboard.
2. Inspect provenance on a wiki page.
3. Jump into the graph.
4. Walk semantically through related concepts.
