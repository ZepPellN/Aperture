# Maps of Content

Maps of Content are living topic maps. They explain how pages relate, where the current tensions are, and what should be read next.

## Core Questions

- Which atomic notes belong to the same topic?
- Which tensions deserve new research?
- Which outputs can be assembled from the current map?

- [[concepts/atomic-notes|Atomic Notes]]
- [[concepts/source-provenance|Source Provenance]]

## Current Judgments

Aperture treats MOCs as coordination surfaces: useful for navigation, agent planning, and deciding when a scattered set of notes has become a real theme.
