# Wiki Absorb Loop

The absorb loop is how Aperture keeps a markdown vault from becoming a junk drawer. Raw material lands in `raw/`, durable claims move into `wiki/`, and one-off reports stay in `outputs/`.

## Loop

```text
raw source
  -> classify the material
  -> update an existing concept or MOC
  -> record provenance and evolution
  -> rebuild indexes and backlinks
  -> let future agents start with context
```

## Upgrade path

The new wiki skills add Zettelkasten-style atomic notes, MOC pages, candidate maturity states, and explicit human-review markers.
