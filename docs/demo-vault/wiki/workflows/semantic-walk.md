# Semantic Walk

Semantic Walk is a deliberately slow exploration mode. Instead of searching for a known keyword, you move from one page to nearby ideas and let the system surface adjacent themes.

## Use cases

- Discover a writing angle.
- Find an unexpected bridge between projects.
- Review a knowledge area without starting from the index.
