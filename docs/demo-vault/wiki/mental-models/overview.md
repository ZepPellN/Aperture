# Mental Models Overview

Mental models help turn raw observations into reusable judgment. Aperture uses them to explain why a note should become an atomic concept, a MOC update, or an output-only draft.

## Useful Models

- File over app.
- Compounding context.
- Capture now, synthesize later.
- Verify before publishing.
