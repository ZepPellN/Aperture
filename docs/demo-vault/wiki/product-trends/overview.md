# Product Trends Overview

This MOC tracks product patterns created by agent-native software: local-first data, human review gates, background automation, and knowledge systems that compound over time.

## Main Tensions

- Capture speed versus knowledge quality.
- Agent autonomy versus explicit human approval.
- Beautiful interfaces versus durable file formats.
