# The Rise of Agentic Coding

Agentic coding refers to the practice of using AI agents to write, review, and maintain software with minimal human intervention. Unlike traditional pair programming or code completion, agentic coding involves delegating entire tasks — from requirement analysis to deployment — to an autonomous agent.

## Key Principles

1. **Task delegation over line-by-line generation**: The human defines the goal; the agent determines the implementation.
2. **Feedback loops**: The agent tests its own code, reads error messages, and iterates.
3. **Context management**: The agent maintains awareness of the broader codebase, not just the current file.

## Leading Tools

- Claude Code
- Cursor Composer
- GitHub Copilot Workspace
- OpenAI Codex

## Implications

This shift represents a fundamental change in software engineering. The human role moves from writing code to designing agent workflows, reviewing architectural decisions, and verifying outcomes.
