# AI Coding Forecasts 2026

This page tracks predictions about the agentic coding landscape in 2026.

## Current Trajectory

Agentic coding tools are moving from experimental to production. Key signals:
- Enterprise adoption of Claude Code and Cursor
- GitHub Copilot Workspace expanding beyond autocomplete
- OpenAI Codex launching as a standalone product

## Open Questions

- How will code review workflows adapt when agents write the majority of code?
- What new security risks emerge from autonomous code generation?
